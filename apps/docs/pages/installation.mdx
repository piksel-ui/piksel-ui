# Installation

You can install Piksel UI to your project using one of the following package managers.

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tabs.Tab>
    
```bash copy
npm install --save @piksel-ui/core @piksel-ui/hooks
```

  </Tabs.Tab>
  <Tabs.Tab>
    
```bash copy
yarn add @piksel-ui/core @piksel-ui/hooks
```

  </Tabs.Tab>
  <Tabs.Tab>
    
```bash copy
pnpm add @piksel-ui/core @piksel-ui/hooks
```

  </Tabs.Tab>
</Tabs>

In order to use Piksel UI components, you need to install these packages
which contain some helper functions, types and hooks. After that, you can go to the
page of the component you want to use and copy the source code to your project.
